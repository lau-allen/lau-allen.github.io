Calendar Web Scraping & API

Web scaping techniques used to retrieve text information from the academic calendar, and Google API to send that information to Google Calendar.

Overview

Web Scraping is a powerful way to retrieve data from webpages to perform data analysis on it. This project demonstrates the process of using Python Requests to retrieve CCNY calendar information using HTTP requests, data processing and formatting using BeautifulSoup, and then apiclient/google_auth_oauthlib to write the data to Google Calendar.

Technical Details: Python (requests, BeautifulSoup, apiclient, google_auth_oauthlib)
